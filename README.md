our first project!

we have 2 prototypes:


1. prototype_1: uses 1 ultrasonic sensor and 1 vibrator.
2. prototype_2: uses 2 ultrasonic sensors and 2 vibrators:

    2.2: usL - left sensor, 
    2.3: vR - right Vibrator, 
    2.4: vL - left Vibrator.
